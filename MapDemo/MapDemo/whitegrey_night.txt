<!--	                                    erlinyou Map Script
	1. Item: 
	    This map style script file is used to set the color and dimension of map element.Each 
	map element has a similar structure:
		<item name="MainRoad">
			<param key="clrBody" value="200;24;32;200;24;32"/>
			<param key="clrBorder" value="136;16;16;136;16;16"/>
			<param key="width" value="22"/>
			<param key="borderWidth" value ="5"/>
			<param key="minWidth" value="2"/>
		</item>
	The item block is embraced by <item name="XXX"> and </item>, the "name" is the unique ID of
	each map item, you not need modify it. 
	
	2. Parameter
		The basic parameter setting is like this: 
		<param key="XXX" value="XXX"/> 
	The "key" is the name of parameter. You don't need to change the value of "key"	, you just 
	change the "value" to set the parameter. 
	    If the parameters is a number ("width", road width) or a string("font", text font),	you 
	set it directly to "value". 
	    If it is a color("clrBody", CoLoR of the road body), the value should be a six-section
	parameter, the first three sections are the RGB (Red, Green, Blue, the value of each color
	is between 0 to 255, this three colors are mixed to get one color in computer, higher value
	means the color has a high proportion in the mixed color) value of the color in DayMode, the
	last three sections are the RGB value of the color in NightMode, each section is separated 
	by ";". Let's take the arrow of path as a example:
	    The origin value is:
		Day Mode: 255;255;0 = Yellow
		Night Mode: 255;255;0 = Yellow
	    If you want to change the day mode color into blue and night mode color into green, the
	parameter will become:
		<param key="clrArrow" value="0;0;255;0;255;0"/>
	    Changing the details of the color parameter, you can get a fantastic map. It's just like
	a painting game, no?

	3. GlobalGroup:
	    This block contains the global elements of the map, such as background color, guidance path,
	guidance arrow, POI list, etc. You can get more information by the commentary line and the value
	of "key" section.
	
	4. ErrorCheck:
	    There is three options for ErroCheck when your value is illegal. If type = 1, a message box 
	will inform you the error and the default value will be used; If type = 2, the default will be 
	used without the message box; If type = 3, the maximum or minimum value will is set according to 
	your value is bigger or smaller than the legal range.
-->
